# Images du Logiciel Aos Renov

Placez ici les 5 captures d'écran du logiciel avec les noms suivants :

1. `generateur-devis.jpg` - Générateur de Devis (formulaire informations entreprise)
2. `estimation-photo.jpg` - Estimation Automatique des Chantiers (upload photo + sélection métier)
3. `dossiers.jpg` - Dossiers (interface de gestion des devis et factures)
4. `dashboard.jpg` - Dashboard (vue d'ensemble avec KPIs et graphiques)
5. `planning.jpg` - Planning des Chantiers (calendrier pour organiser les interventions)

## Format recommandé
- Format : JPG ou PNG
- Taille : 1920x1080px minimum (ou ratio 16:10)
- Poids : < 500KB par image pour optimiser le chargement

Les images seront affichées dans un carousel interactif dans la section "Ce que l'artisan découvrira".
